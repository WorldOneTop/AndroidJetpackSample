# Room + Retrofit
Android Jetpack Library : Room

Room 지속성 라이브러리는 SQLite를 완벽히 활용하면서 원활한 데이터베이스 액세스가 가능하도록 SQLite에 추상화 계층을 제공합니다.

Retrofit은 네트워크 관련 라이브러리입니다.
